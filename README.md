# Demo Store 🛍️

A headless e-commerce store built with Next.js, Shopify, and Builder.io. High-performance, SEO-optimized, and fully customizable.

## ✨ Overview

This is a modern headless e-commerce solution combining Shopify's powerful e-commerce backend with Builder.io's visual CMS and Next.js for ultra-fast performance.

## 🛠️ Tech Stack

- **Next.js** - React framework for production
- **Shopify** - E-commerce backend
- **Builder.io** - Visual CMS
- **Tailwind CSS** - Styling

## 🔋 Key Features

- ✅ Ultra high performance
- ✅ SEO optimized
- ✅ Visual CMS with Builder.io
- ✅ Shopify product sync
- ✅ Themable and customizable
- ✅ Responsive design

## 🚀 Quick Start

### Prerequisites
- Node.js installed
- npm or yarn
- Shopify account
- Builder.io account
